# Odin Testing

This project was created for the Odin Project with the purpose of creating software with Test Driven Development and using Jest.

## This project contains

- A function to capitalize the first character of a string and test cases
- A function to reverse a string and test cases
- A calculator object encapsulated in an IIFE that contains methods for adding, subtracting, multiplying and dividing, and test cases
- A caesar cipher function that shifts letters in a string based on a shift factor, and test cases
- An analyzeArray object that takes an array of number and contains properties such as average, min, max, and length, and test cases

## Concepts used for this project

- Test Driven Development (TDD)
- Jest and matchers
- Using ESM imports with Jest through Babel
- npm, package.json
- ESLint, Prettier
